# FishNet
Simple social network for fishermans

# Prerequisites
 * NodeJS
 * .NET 5.0 SDK
 * Android Studio
 
# Steps to launch application
 * Create new virtual device and run it (https://reactnative.dev/docs/environment-setup)
 * Go to ClientApp folder and run "npm install"
 * Go to API/API folder and run "dotnet run ./API.csproj"
 * Go to ClientApp folder and run "react-native run-android"
 
 (If you are a Windows user, you can run "launch.cmd" file instead of completing the last two steps)
